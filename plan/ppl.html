Push / Pull / Legs Trainingsplan

Dieser Push / Pull / Legs Plan besteht aus vier Trainingstagen. Drei Einheiten bilden das klassische PPL-Gerüst, der vierte Tag ist als flexible Zusatz- oder Fokus-Einheit gedacht. Alle Inhalte sind Platzhalter und können später durch deinen eigenen Plan ersetzt werden.

Level Platzhalter – frei anpassbar
Einheiten pro Woche 4 Einheiten (z. B. P / P / L / Bonus)
Trainingsziel Platzhalter (z. B. Muskelaufbau, Kraft)
Empfohlene Dauer mindestens 12 Wochen

Deine PPL-Woche

Dieser Plan nutzt eine klassische Push / Pull / Legs Struktur mit einem zusätzlichen vierten Trainingstag. Alle Übungen und Sätze sind Platzhalter und können später individuell gefüllt werden.

Tag 1

Push

Fokus: Brust, vordere Schultern und Trizeps.

  • Push Übung 1 (Platzhalter) Sätze / Wdh.
  • Push Übung 2 (Platzhalter) Sätze / Wdh.
  • Push Übung 3 (Platzhalter) Sätze / Wdh.
  • Push Übung 4 (Platzhalter) Sätze / Wdh.
Tag 2

Pull

Fokus: Rücken, hintere Schultern und Bizeps.

  • Pull Übung 1 (Platzhalter) Sätze / Wdh.
  • Pull Übung 2 (Platzhalter) Sätze / Wdh.
  • Pull Übung 3 (Platzhalter) Sätze / Wdh.
  • Pull Übung 4 (Platzhalter) Sätze / Wdh.
Tag 3

Legs

Fokus: Unterkörper und Core.

  • Legs Übung 1 (Platzhalter) Sätze / Wdh.
  • Legs Übung 2 (Platzhalter) Sätze / Wdh.
  • Legs Übung 3 (Platzhalter) Sätze / Wdh.
  • Legs Übung 4 (Platzhalter) Sätze / Wdh.
Tag 4

Bonus / Fokus

Flexible Einheit: z. B. Arme, Technik, Schwachstellen oder eine leichte Full Body Session.

  • Bonus Übung 1 (Platzhalter) Sätze / Wdh.
  • Bonus Übung 2 (Platzhalter) Sätze / Wdh.
  • Bonus Übung 3 (Platzhalter) Sätze / Wdh.
  • Bonus Übung 4 (Platzhalter) Sätze / Wdh.

Hinweise zum PPL-Plan

Alle Inhalte sind Platzhalter. Du kannst hier später erklären, wie dein PPL-Plan aufgebaut ist, wie du Belastung und Pausen verteilst und welche Varianten es für unterschiedliche Ziele gibt.

  • Hinweis 1 (Platzhalter): Wie viele Wochen der Plan am Stück laufen soll.
  • Hinweis 2 (Platzhalter): Welche Satz- und Wiederholungsbereiche sinnvoll sind.
  • Hinweis 3 (Platzhalter): Hinweise zu Deload-Wochen oder optionalen Pausentagen.

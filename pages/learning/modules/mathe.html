Matrizen

Matrizen sind rechteckige Anordnungen von Zahlen, Symbolen oder Funktionen in Zeilen und Spalten. Sie werden in Mathematik, Informatik und Physik eingesetzt, z. B. zur Darstellung von linearen Gleichungssystemen, Transformationen oder zur Datenorganisation.

Grundbegriffe

  • Dimensionen: Anzahl der Zeilen und Spalten, z. B. 2x3-Matrix.
  • Einträge: Die einzelnen Elemente der Matrix, z. B. a11, a12 …
  • Quadratische Matrix: Gleiche Anzahl an Zeilen und Spalten.
  • Diagonalmatrix: Nur die Diagonaleinträge ungleich null, alle anderen = 0.

Beispiel

A = | 1️⃣ 2️⃣ 3️⃣ |
    | 4️⃣ 5️⃣ 6️⃣ |
    | 7️⃣ 8️⃣ 9️⃣ |

Operationen

  • Addition: Elementweise Addition gleich großer Matrizen.
  • Skalarmultiplikation: Jede Komponente wird mit einer Zahl multipliziert.
  • Multiplikation: Zeilen der ersten Matrix × Spalten der zweiten Matrix (Anzahl Spalten erste = Zeilen zweite).
  • Determinante: Prüft z. B. die Invertierbarkeit (nur quadratisch).
  • Inverse: A⁻¹, sodass A * A⁻¹ = I (Einheitsmatrix).
  • Transponieren: Zeilen und Spalten vertauschen → AT.

Limes

Der Limes beschreibt das Verhalten einer Funktion, wenn die Eingabe einem bestimmten Wert, 0 oder ∞ näher kommt. Er ist zentral in der Analysis und bei der Berechnung von Ableitungen.

Limes gegen einen Wert

f(x) = (x² - 1️⃣)/(x - 1️⃣)
lim x→1️⃣ f(x) = 2️⃣

Limes gegen 0

f(x) = sin(x)/x
lim x→0️⃣ f(x) = 1️⃣

Limes gegen ∞

f(x) = 1️⃣/x
lim x→∞ f(x) = 0️⃣

Hinweise

  • Wenn der Limes existiert, nähert sich die Funktion einem festen Wert.
  • Bei unendlichen Grenzwerten spricht man von Divergenz.
  • Oft hilfreich: Faktorisieren, kürzen oder bekannte Grenzwertregeln anwenden.

Übungsaufgaben

Vollständige Induktion


Beweise durch vollständige Induktion, dass folgende Aussage gilt:

2️⃣ + 4️⃣ + 6️⃣ + ... + 2️⃣n = n(n + 1️⃣)
1️⃣ + 3️⃣ + 5️⃣ + ... + (2️⃣n - 1️⃣) = n²

Matrizen

Gegeben sind die Matrizen:

A = | 1️⃣ 2️⃣ |
    | 3️⃣ 4️⃣ |

B = | 5️⃣ 6️⃣ |
    | 7️⃣ 8️⃣ |

Aufgaben:

1️⃣ Berechne die Summe A + B
2️⃣ Berechne das Produkt A * B
3️⃣ Bestimme die Determinante von A
4️⃣ Transponiere die Matrix B (B^T)

Limes

Berechne den Grenzwert der folgenden Funktionen:

1️⃣ lim x→0️⃣ (sin(x) / x)
2️⃣ lim x→∞ (1️⃣ + 1️⃣/x)^x
3️⃣ lim x→2️⃣ ((x² - 4️⃣) / (x - 2️⃣))
